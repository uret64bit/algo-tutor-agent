# Checklist

## Phase 1: 项目基础
- [ ] 前端项目可通过 `npm run dev` 正常启动
- [ ] 后端项目可通过 `uvicorn` 正常启动
- [ ] Docker Compose 一键启动所有服务（PostgreSQL, Redis, 后端, 前端）
- [ ] 用户可注册、登录、获取 JWT Token
- [ ] 用户角色区分（学员/教练/管理员），权限控制生效
- [ ] 教练可创建小组、生成邀请码
- [ ] 学员可通过邀请码加入小组
- [ ] 教练可查看小组成员列表

## Phase 2: 内容与知识库
- [ ] 知识点数据模型正确（含前置依赖、难度等级）
- [ ] 知识点图谱前端可展示 DAG 依赖关系
- [ ] 种子知识点数据已预置（数组、链表、栈、队列、二分、DP、图论等）
- [ ] 讲义支持三级粒度（知识卡片/标准讲义/深度专题）
- [ ] 模板代码可关联知识点，支持多语言版本
- [ ] 讲义内容可切片、向量化并存入向量数据库
- [ ] AI 问答可检索知识库并生成带引用来源的回答
- [ ] 题目数据模型正确（含知识点标签、难度、测试用例）
- [ ] 题目可按知识点和难度筛选
- [ ] 种子题目数据已预置（每知识点 3-5 道）

## Phase 3: 核心学习
- [ ] 代码提交后可在 Docker 沙箱中编译运行
- [ ] 支持 C++、Java、Python 三种语言
- [ ] 判题结果正确（AC/WA/TLE/RE）
- [ ] 判题后给出时间/空间复杂度分析
- [ ] 渐进式提示按层次展示（思路 → 关键观察 → 伪代码）
- [ ] 代码 Review 包含命名、边界、可优化点分析
- [ ] 最优解对比展示正确
- [ ] 学习记录自动采集（做题记录、知识点状态、打卡）
- [ ] 进度面板展示正确指标（完成数、通过率、打卡天数、雷达图）
- [ ] 弱项诊断正确（通过率 < 60% 自动标记）
- [ ] 错题自动收录（WA/TLE/RE 触发）
- [ ] 错题本可查看并推荐同类题

## Phase 4: 智能教学
- [ ] 知识点完成后自动解锁下一知识点
- [ ] 连续出错 3 次触发补漏推送
- [ ] 教-练-测闭环流程完整可走通
- [ ] 遗忘临界点计算正确（1d/2d/4d/7d/15d/30d）
- [ ] 定时任务在临界点触发复习提醒
- [ ] 复习完成后遗忘曲线阶段正确更新
- [ ] 前端可查看待复习列表和复习卡片
- [ ] 变体生成结果正确（同知识点、同难度、不同场景）
- [ ] 变体题可收藏并直接练习

## Phase 5: 协作与激励
- [ ] 题解支持 Markdown 发布/编辑/删除
- [ ] 评论和点赞功能正常
- [ ] 点赞 >= 10 自动标记为精选题解
- [ ] 排行榜正确展示小组内和全局排名
- [ ] 成就徽章触发条件正确，自动颁发
- [ ] 每日打卡记录正确，连续天数计算正确
- [ ] 模拟面试限时 45 分钟，随机抽题 2-3 道
- [ ] 面试结束后生成面试报告
